# Sistema Universidad - C++ con SQL Server

## Descripción

Aplicación de consola en C++ conectada a SQL Server mediante ODBC. Permite gestionar alumnos, cursos y secciones.

## Funcionalidades

* Conexión a base de datos
* Menú interactivo
* CRUD de alumnos, cursos y secciones
* Asignación de cursos y secciones
* Consulta JOIN
